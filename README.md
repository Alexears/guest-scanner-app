# Guest Scanner App

Чистовая версия v13.

## Что изменено

- один основной экран: сканер + результат + ручной поиск + статус сервера
- отдельный второй экран: `display.html`
- сканер после успешного считывания останавливается
- для повторного запуска используется кнопка `Новое сканирование`
- сканируются только коды от `001` до `999`
- поиск по коду работает только по кнопке `Найти`
- если введён код, поиск идёт только по коду
- если введены имя и/или фамилия:
  - один результат — сразу карточка
  - несколько результатов — список выбора
- кнопка `Убрать с экранов` отправляет заглушку `000`

## Локальный запуск

```bash
python3 server.py
```

- добавлена кнопка вспышки для устройств, где браузер поддерживает torch

- кнопка вспышки теперь всегда видна во время сканирования
- если torch не поддерживается, по нажатию показывается понятное сообщение
