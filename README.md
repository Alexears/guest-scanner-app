# Guest Scanner App

Готовый репозиторий для загрузки на GitHub и деплоя на Railway.

## Что внутри

- `index.html` — экран сканера
- `admin.html` — админка
- `display.html` — экран вывода карточки
- `server.py` — простой Python-сервер и API
- `guests.xlsx` — пример Excel
- `001.jpg`, `002.jpg`, `003.jpg` — пример карточек
- `state.json` — текущее состояние приложения

## Локальный запуск

```bash
python3 server.py
```

Открыть:
- `http://localhost:8000/index.html`
- `http://localhost:8000/admin.html`
- `http://localhost:8000/display.html`

## Деплой на Railway через GitHub

1. Создай новый репозиторий на GitHub.
2. Загрузи в него все файлы из этой папки.
3. Зайди в Railway.
4. Нажми **New Project**.
5. Выбери **Deploy from GitHub Repo**.
6. Выбери этот репозиторий.
7. Railway сам поднимет приложение.

## Важно

- Картинки должны называться строго по коду: `001.jpg`, `002.jpg`, и так далее.
- Для продакшена лучше потом вынести состояние и базу из `state.json` / Excel в нормальную БД.
- Сейчас это рабочий вариант для ивента и быстрого запуска.
