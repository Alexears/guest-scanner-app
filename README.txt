Запуск на Mac:
1. Распакуй архив.
2. В Terminal:
   cd ~/Downloads/guest_scanner_server_app_v6
   python3 server.py

Открыть:
- http://localhost:8000/index.html   — сканер
- http://localhost:8000/admin.html   — админка
- http://localhost:8000/display.html — экран

Что изменено:
- весь интерфейс теперь в тёмной теме
- в админке слева поиск и управление
- справа большая карточка гостя
